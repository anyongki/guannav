# Typecho 导航主题 - 使用示例

本文档提供了详细的使用示例，帮助您快速上手本主题。

## 示例场景

我们假设要创建一个"二次元导航"网站，包含动漫、漫画、下载等分类。

## 第一步：创建分类

### 1. 访问分类管理

在 Typecho 后台：
1. 点击 **管理**
2. 选择 **分类**

### 2. 创建分类

创建以下分类：

| 分类名 | 描述 | 示例 |
|--------|------|------|
| 动漫 | 动漫相关网站 | AniCh、Animeko 等 |
| 漫画 | 漫画相关网站 | heibaizhan、Animoe 等 |
| 下载 | 下载资源网站 | 西瓜卡通、风铃动漫 等 |
| 神器 | 实用工具网站 | 各类在线工具 |
| 影视 | 影视相关网站 | 动画电影、剧集等 |
| 音乐 | 音乐相关网站 | 音乐播放器、歌曲库等 |
| 阅读 | 阅读相关网站 | 轻小说、漫画阅读等 |
| 游戏 | 游戏相关网站 | 游戏下载、攻略等 |
| 娱乐 | 娱乐相关网站 | 其他娱乐资源 |

## 第二步：添加网站导航

### 示例 1：添加 AniCh 动漫弹幕网

**在后台操作：**

1. 点击 **管理** → **写文章**

2. 填写基本信息：
   - **文章标题**：`AniCh动漫弹幕网`
   - **文章分类**：选择 **动漫**
   - **文章内容**：（可选）
     ```
     AniCh是一个高清片源的追番平台，支持弹幕追番。
     提供最新的日本动漫资源，支持多个播放源。
     ```

3. 向下滚动，找到 **导航主题字段** 部分

4. 填写自定义字段：
   - **网站 URL**：`https://www.anichi.com`
   - **阿里图标代码**：`icon-play`（可选）
   - **平台标识**：`Web, iOS, Android`
   - **网站简介**：`高清片源的追番平台，支持弹幕追番`

5. 点击 **自动获取图标** 按钮
   - 系统会自动获取网站的 Favicon
   - 预览框中会显示获取到的图标

6. 点击 **发布** 保存

**前台效果：**

卡片会显示为：
```
┌─────────────────────────────┐
│ [Favicon] AniCh动漫弹幕网 [♪]│
│                             │
│ 高清片源的追番平台，支持    │
│ 弹幕追番                    │
│                             │
│ Web  iOS  Android      [→]  │
└─────────────────────────────┘
```

## 第三步：配置阿里 Iconfont

### 1. 创建 Iconfont 项目

1. 访问 [Iconfont](https://www.iconfont.cn/)
2. 注册/登录账号
3. 创建新项目，命名为 "二次元导航"
4. 添加以下图标：
   - `icon-play` - 播放图标
   - `icon-star` - 星星图标
   - `icon-download` - 下载图标
   - `icon-heart` - 心形图标
   - `icon-arrow` - 箭头图标

### 2. 获取 CDN 链接

1. 在 Iconfont 项目页面，找到 **生成链接** 或 **复制代码**
2. 复制 CSS 链接，例如：
   ```
   https://at.alicdn.com/t/font_1234567890_abcdefg.css
   ```

### 3. 配置主题

1. 编辑 `/usr/themes/typecho-nav-theme/index.php`
2. 找到第 11 行左右的 link 标签：
   ```html
   <link rel="stylesheet" href="https://at.alicdn.com/t/font_1234567890_abcdefg.css">
   ```
3. 替换为您的 Iconfont CDN 链接
4. 保存文件

## 第四步：查看前台效果

### 访问网站

1. 在浏览器中访问您的 Typecho 网站
2. 您应该看到：
   - 左侧导航栏，显示所有分类
   - 中间是卡片网格，显示所有添加的网站
   - 每个卡片显示网站图标、名称、简介、平台标识等

### 卡片交互

- **悬停卡片**：卡片会有轻微的上升动画和边框颜色变化
- **点击箭头按钮**：会跳转到对应的网站
- **点击分类**：会筛选显示该分类下的网站

## 高级技巧

### 1. 修改卡片宽度

编辑 `style.css`，找到 `.cards-grid` 部分：

```css
.cards-grid {
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
}
```

修改 `280px` 为您想要的宽度：
- 更小（如 250px）：卡片更紧凑
- 更大（如 320px）：卡片更宽松

### 2. 修改主题颜色

编辑 `style.css`，找到 `:root` 部分，修改颜色变量：

```css
:root {
    --color-accent: #4a9eff;  /* 改为您想要的颜色 */
}
```

常用颜色代码：
- `#4a9eff` - 蓝色（默认）
- `#ff6b6b` - 红色
- `#51cf66` - 绿色
- `#ffd93d` - 黄色
- `#a78bfa` - 紫色

### 3. 修改左侧导航栏宽度

编辑 `style.css`，找到 `.sidebar` 部分：

```css
.sidebar {
    width: 200px;  /* 改为您想要的宽度 */
}
```

### 4. 隐藏左侧导航栏

编辑 `style.css`，修改 `.sidebar` 部分：

```css
.sidebar {
    display: none;
}
```

## 故障排除

### 问题：自动获取 Favicon 不工作

**检查清单：**
1. ✓ 确认网站 URL 格式正确（包含 http:// 或 https://）
2. ✓ 确认目标网站可以正常访问
3. ✓ 检查服务器是否允许外部 HTTP 请求
4. ✓ 尝试手动输入 Favicon URL

**手动获取 Favicon：**
1. 访问 `https://www.google.com/s2/favicons?sz=128&domain=example.com`（将 example.com 替换为目标网站）
2. 复制图片 URL
3. 在后台粘贴到 Favicon 字段

### 问题：图标不显示

**检查清单：**
1. ✓ 确认 Iconfont CDN 链接正确
2. ✓ 确认图标代码正确（如 `icon-star`）
3. ✓ 检查浏览器控制台是否有错误

**调试方法：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签中是否有错误信息
3. 查看 Network 标签中 Iconfont CSS 是否正确加载

### 问题：卡片显示不正常

**检查清单：**
1. ✓ 清除浏览器缓存（Ctrl+Shift+Delete）
2. ✓ 确认 style.css 正确上传
3. ✓ 检查主题文件夹权限（应为 755）

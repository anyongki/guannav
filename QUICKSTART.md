# Typecho 导航主题 - 快速开始指南

5 分钟快速上手本主题！

## 安装（1 分钟）

### 1. 下载主题

从 GitHub 下载或解压主题文件。

### 2. 上传到服务器

将 `typecho-nav-theme-php` 文件夹上传到 `/usr/themes/` 目录。

最终路径应为：`/usr/themes/typecho-nav-theme-php/`

### 3. 在后台启用

1. 登录 Typecho 后台
2. 进入 **设置 → 外观**
3. 找到 **导航主题**
4. 点击 **启用**

✅ 主题已启用！

## 配置（2 分钟）

### 1. 创建分类

在后台 **管理 → 分类** 中创建以下分类：

- 动漫
- 漫画
- 下载
- 神器
- 影视
- 音乐
- 阅读
- 游戏
- 娱乐

### 2. 配置 Iconfont（可选）

1. 访问 [Iconfont](https://www.iconfont.cn/)
2. 创建项目并添加图标
3. 复制 CDN 链接
4. 编辑 `/usr/themes/typecho-nav-theme-php/index.php` 第 11 行，替换为您的链接

## 添加网站（2 分钟）

### 1. 新建文章

在后台 **管理 → 写文章**

### 2. 填写信息

- **标题**：网站名称（如 "AniCh动漫弹幕网"）
- **分类**：选择对应分类
- **内容**：可选

### 3. 填写自定义字段

向下滚动找到 **导航主题字段** 部分：

| 字段 | 示例 | 说明 |
|------|------|------|
| 网站 URL | https://www.anichi.com | 必填 |
| 阿里图标代码 | icon-star | 可选 |
| 平台标识 | iOS, Android, Web | 可选 |
| 网站简介 | 高清片源的追番平台 | 可选 |

### 4. 自动获取图标

1. 点击 **自动获取图标** 按钮
2. 系统会自动获取网站 Favicon
3. 预览框中显示获取结果

### 5. 发布

点击 **发布** 保存

✅ 网站已添加！

## 查看效果

访问您的网站首页，您应该看到：

- ✅ 左侧分类导航栏
- ✅ 中间卡片网格
- ✅ 每个卡片显示网站图标、名称、简介、平台标识
- ✅ 可以点击卡片右下角的箭头跳转到网站

## 常见问题

### Q: 自动获取图标不工作？

A: 检查以下几点：
1. URL 格式是否正确（如 https://example.com）
2. 目标网站是否可以正常访问
3. 服务器是否允许外部请求

### Q: 图标不显示？

A: 检查以下几点：
1. Iconfont CDN 链接是否正确
2. 图标代码是否正确（如 icon-star）
3. 浏览器缓存是否需要清除

### Q: 如何修改颜色？

A: 编辑 `/usr/themes/typecho-nav-theme-php/style.css`，修改 `:root` 中的颜色变量。

### Q: 如何修改卡片宽度？

A: 编辑 `style.css`，找到 `.cards-grid` 部分，修改 `minmax(280px, 1fr)` 中的 280px。
